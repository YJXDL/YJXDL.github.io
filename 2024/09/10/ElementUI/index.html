Element UI简介

基于vue的一套桌面端组件，提前封装好的UI模板。方便开发者快速搭建一个网站前端界面。
官网：https://element.eleme.cn/

Element UI安装

首先创建Vue项目

  • 确保电脑安装Vue环境
  • 执行命令 vue ui 打开浏览器打开Vue管理界面
  • 打开小房子按钮
  • 选择创建
  • 设置项目存放路径
  • 点击在此创建项目
  • 输入项目名称，点击下一步
  • 选择手动配置项目，点击下一步
  • 进行功能配置，点击下一步
  • 打开hostory mode,点击创建项目
  • 点击创建项目不保存预设
  • 等待项目创建结束
  • 出现该图片代表项目安装成功

    给项目安装ElementUI插件

  • 点击插件
  • 点击添加插件
  • 搜索框输入element,选中第一项
  • 点击完成安装
  • 点击继续就会发现ElementUI插件已经安装在项目中，界面如下所示

    启动项目

  • 打开项目，使用Ideal打开
  • 打开之后可以看见Plugin中有elelement说明element已经成功安装在Vue项目中
  • 终端执行命名 npm run serve
  • 若安装成功出现以下界面
